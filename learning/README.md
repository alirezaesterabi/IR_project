# Learning Materials

This folder contains the theory-first learning material for the Information Retrieval project.

It is intentionally separate from the implementation folders such as `notebooks/`, `scripts/`, and `src/`.

## Structure

```text
learning/
├── README.md                          # This entry point
├── modules/                           # Curated learning modules
└── raw_materials/                     # Course slides, labs, and source materials
```

## What To Read First

1. Start here in `learning/README.md`
2. Use `learning/modules/README.md` for the module overview
3. Study the module folders under `learning/modules/`
4. Use `learning/raw_materials/` as supporting course material, not the main entry point

## Folder Roles

- `modules/` contains the structured theory and notebook material
- `raw_materials/` contains lecture slides, lab notebooks, and other reference files

## Relationship To The Project

- `learning/` is for understanding concepts deeply
- `notebooks/` and `src/` are where the project implementation lives
- The idea is: learn concepts here, then apply them in the project
